移动文件
```
sudo mv 被移动文件 移动位置
```
删除文件
```
sudo rm
```
删除文件夹
```
sudo rm -rf 
```
解压压缩包
```
sudo tar -xf
```
建立软连接
```
sudo ln -s 被建立软连接文件绝对路径 软连接创建位置
```
进入root
```
sudo su
```
创建文件
```
touch
```
创建文件夹
```
mkdir
```
文件重命名
```
mv old_file.txt new_file.txt
```
